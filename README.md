# 📊 Data Visualization Project (CodeAlpha Internship)

This project is part of my Data Analytics Internship at CodeAlpha.

## 🚀 Project Overview
In this project, I performed data visualization on a dataset of 500+ products collected from an e-commerce website. The goal was to transform raw data into meaningful insights using visual techniques.

## 📂 Folder Structure
CodeAlpha_DataVisualization/
│── books_data_500.csv
│── cleaned_books_data.csv
│── visualization.py
│── screenshots/

## 📊 Visualizations Created
- Price Distribution of Products
- Rating Distribution
- Price vs Rating Analysis
- Top 10 Most Expensive Products
- Top 10 Cheapest Products
- Average Price by Rating
- Availability Analysis
- Correlation Heatmap

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Key Insights
- Identified price distribution patterns
- Analyzed rating trends across products
- Found high-value and low-cost products
- Understood relationship between price and ratings

## 📁 Output
- Cleaned dataset generated
- Multiple visual insights extracted

## 📌 Conclusion
This project helped me understand the importance of data visualization in data analysis and how visual storytelling can simplify complex datasets.
